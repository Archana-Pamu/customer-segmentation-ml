# Customer Segmentation Dashboard

## Project Overview

Customer Segmentation Dashboard is a web-based application developed using Python Flask and Machine Learning. The project analyzes customer behavior and groups customers into different segments using the K-Means Clustering algorithm.

## Features

* Customer Segmentation using K-Means Clustering
* Interactive Dashboard
* 3D Cluster Visualization
* Elbow Method Analysis
* Feature Correlation Heatmap
* Income vs Spending Analysis
* Age Distribution Analysis
* Bulk Excel File Upload and Processing
* Customer Recommendations Based on Cluster

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Plotly
* HTML/CSS

## Dataset

The project uses the Mall Customers dataset containing:

* Customer ID
* Gender
* Age
* Annual Income
* Spending Score

## How to Run

1. Install required libraries:

   ```bash
   pip install flask pandas numpy matplotlib seaborn plotly scikit-learn openpyxl
   ```

2. Run the application:

   ```bash
   py app.py
   ```

3. Open the browser and visit:

   ```
   http://127.0.0.1:5000
   ```
